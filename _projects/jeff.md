<h2>Summary</h2>
<p><b><a href="https://maxy3702.itch.io/jeff">Jeff</a></b> is a short narrative game I made over <b>three days</b> for a game jam my the uOttawa Game Dev Club was hosting. The theme was <b>'it started with a note'</b>. The game revolves around a guy named Jeff who's planning to dress up as famous creepypasta character Jeff the Killer for his friends' halloween party.</p>
<p>This game does require some surface knowledge of who Jeff the Killer is in order to appreciate it, but I think it generally stands on its own merit.</p>

<h2>Notable Features</h2>
* The aesthetic. For some reason, I decided to theme the whole game around this schoolwork stationary, paper-y, crayon-y aesthetic. Although it doesn't make that much sence with the narrative, I really think I executed it beautifully.
* The dialogue system looks really good and functions pretty well.
